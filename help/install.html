Installation instructions for New Install

  1. Install kshop as a standard module.
  2. CHMOD 664 the directory kshop/images/products. If you still can't upload images then try CHMOD 777
  3. CHMOD 664 file kshop/libclasses/xajax/xajax_js/xajax.js. If the orders menu or plugin menu still do not work correctly try CHMOD 777. This step might not be necessary for all servers.
  4. Go into kshop preferences and set all your options such as email etc. Keep in mind that if "require login" is set to yes, anonymous users will NOT be allowed to purchase goods using kshop.
  5. Goto plugins menu in kshop administration and install at least 1 payemnt and 1 shipping plugin.
  6. ERASE update.php file!

 

Instalation Instructions for upgrade

1. overwrite your 2.03 files with the new 2.2 files
2. run update.php
3. Update module
3. ERASE update.php file!

 

 

That's it! kshop should now work for you!
